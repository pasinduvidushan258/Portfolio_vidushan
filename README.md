## ✨ Features Breakdown

- **Dynamic Content:** Full CRUD (Create, Read, Update, Delete) management for projects and services via the Admin Dashboard.
- **Robust Security:** PDO prepared statements and extensive SQL injection protection, ensuring data integrity.
- **Optimized for Search:** Direct `robots.txt` and `noindex` configurations to hide sensitive admin pages and focus on public content.
- **Glass-morphism Design:** A modern header and responsive UI that looks sharp on everything from phones to desktops.

## Local Setup Instructions

1. **Clone:** `git clone https://github.com/pasinduvidushan/Portfolio_vidushan.git`
2. **Move:** Copy to local `htdocs` or `www` directory.
3. **Database:** Import the provided `.sql` file to your MySQL server.
4. **Configure:** Update credentials in `includes/db_connect.php`.
5. **Run:** Access at `http://localhost/Portfolio_vidushan`.
